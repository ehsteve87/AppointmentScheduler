Application Title: Application Scheduler

Application Purpose: Keeps track of scheduled appointments for customers, contacts, and users.

Author: Steve Decker, stevenmitchelldecker@gmail.com

Application version: 1.0

Date: February 21, 2022

IDE: IntelliJ IDEA Community Edition 2021.1.1 x64

JDK: Java SE 17.0.1

JavaFX: JavaFX-SDK-17.0.1

How to run the program: In JDBC.java, change the class variables to match your own MySQL database (DDL not provided here), then run Main.main(args).
The database must have at least one user in the Users table before the application is launched.

Additional report: Calculates the average length in minutes of all appointments of a given type.

MySQL Connector driver: mysql-connector-java-8.0.25
